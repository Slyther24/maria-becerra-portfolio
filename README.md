# maria-becerra-portfolio

## Portafolio web HTML + CSS + JS

Este proyecto contiene un portafolio web estatico para mostrar proyectos y
publicarlo en GitHub Pages.

### Archivos principales

- `index.html`: estructura y contenido del portafolio.
- `styles.css`: estilos visuales, diseno responsive y paleta de colores.
- `script.js`: lista de proyectos y filtros por categoria.

### Paleta de colores

- Blanco: `#ffffff`
- Vinotinto: `#2e000f`
- Gris oscuro: `#33312f`

### Publicacion en GitHub Pages

1. Sube estos archivos al repositorio.
2. En GitHub, entra a `Settings`.
3. Abre `Pages`.
4. Selecciona la rama principal y la carpeta raiz.
5. Guarda los cambios.
